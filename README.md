**e107 2.3.4** – Le CMS open source léger, rapide et hautement personnalisable pour créer des sites web dynamiques.

### Points forts de cette version
- CMS ultra-léger et performant
- Système de thèmes et plugins très flexible
- Interface d’administration moderne
- Support multilingue et multi-utilisateurs
- Idéal pour sites communautaires, portails, blogs et intranets
- Développement actif et communauté francophone forte
